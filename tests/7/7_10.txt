Бинарное дерево представляет из себя структуру данных, у каждого элемента которого есть до двух потомков: левый и правый.  
Один из способов хранить их: список потомков левого и правого. 
Считаем "корнем" ноль, и если потомок имеет номер - 0, то потомка нет.

Листом называют вершину дерева без потомков. Посчитать количество листов в дереве. 
```
l = [9, 0, 4, 7, 0, 1, 0, 0, 0, 0]
r = [5, 0, 6, 8, 0, 2, 0, 0, 0, 3]
```
