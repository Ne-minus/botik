Напишите функцию `splitter`, которая принимает на вход две строки, `text` и `s`. Функция возвращает список `t`, разбив `text` на части с разделением по `s` (например, `s` может иметь значение `'\n'`, `', '` и т.д.).
Обратите внимание, что если разделитель `s` не указан, то по умолчанию текст делится по пробелу.
```
>>> splitter('Мама мыла раму', ' ')
['Мама', 'мыла', 'раму']
>>> splitter('Мама мыла раму', 'а')
['М', 'м', ' мыл', ' р', 'му']
>>> splitter('Мама мыла раму', 'м')
['Ма', 'а ', 'ыла ра', 'у']
```
