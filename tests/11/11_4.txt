У вас есть файл с лакским словарем (https://gist.github.com/Pandaklez/3f93c878d4e18586d5b22150fd6498dc). Обратите внимание, что у некоторых слов, есть несколько переводов, которые стоят под арабскими цифрами. Сделайте словарь, где ключом является слово, а значением является словарь с ключом "перевод", где значения - это список переводов. Это задание можно объединить с предыдущим, если вы будете объединять с предыдущим то ожидается такой словарь:

```
{'анализ': {'грамм.тег': 'м', 
            'окончание': '',
            'перевод': ['хъирив лаяву, лух1и-к1яла лич1и баву', 'анализ (оьттул)']
          }
}
```
