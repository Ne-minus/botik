# Задача 1.

Напишите конвертер из `json` в `csv`. 

**В вашем коде должна быть функция, названная `json2csv`**, которая 
первым аргументом принимает путь к существующему `json`-файлу,
вторым аргументом принимает путь к желанному `csv`-файлу,
и в результате работы которой в указанном файле появляется `csv`-содержимое, соответствующее переданному `json`.

`json`-файл гарантировано содержит список словарей (в `json`-смысле этих слов). Например,
```json
[
  {
    "name": "кошка",
    "хвост": "пушистый"
  },
  {
    "name": "пёс",
    "хвост": "крючком"
  },
  {
    "name": "стол",
    "цвет": "синий"
  }
]
```

**Примечания:**
* `json` не содержит вложенных объектов.   
* пустой `json`-файл должен превращаться в пустой `csv`-файл.
* в csv должна найтись колонка для каждого из встреченных ключей.  
  то, что у стола нет хвоста, должно привести к тому, что в строке про стол в колонке хвост будет ничего.
