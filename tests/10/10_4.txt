# Задача 4.

Дан `.md` файл в формате 
```markdown
# заголовок 
text1
## подзаголовок
text2
## подзаголовок
text3

# ещё заголовок
## ещё подзаголовок
### и ещё
text4
```
Пожалуйста, научитесь превращать его (и ему подобных) в `json`, где вложенность заголовков &mdash; это вложенность документов.  

формат документа: 
```json
[
    {
        "title": "заголовок",
        "text": "text1",
        "children": [
            {
                "title": "подзаголовок",
                "text": "text2",
                "children": []
            },
            {
                "title": "подзаголовок",
                "text": "text3",
                "children": []
            }
        ]
    },
    {
        "title": "ещё заголовок",
        "text": "",
        "children": [
            {
                "title": "ещё подзаголовок",
                "text": "",
                "children": [
                    {
                        "title": "и ещё",
                        "text": "text4",
                        "children": []
                    }
                ]
            }
        ]
    }
]
```

**Ваше решение должно содержать функцию, названную `md2json`**. Эта функция должна принимать два аргумента: 
1. имя `.md`-файла, который надо превратить в `json`
2. имя файла, в который должен быть записан результирующий `json`.

**Примечание:** 
* Игнорируйте пустые строки.
* Вложенность определяется количеством решёток.
